ml313
=====

Python toolbox for statistical analysis and machine learning.

Installation
------------

Dependencies
~~~~~~~~~~~~

ml313:

- Python (>= 3.5)
- NumPy
- SciPy
- Matplotlib
- Pandas
- scikit-learn
- StatsModels
- Joblib
- scikits.bootstrap
- imbalanced-learn
- XGBoost

Installation from sources
~~~~~~~~~~~~~~~~~~~~~~~~~

Clone the repository and install it with ``pip``::

    git clone https://github.com/hubertgabrys/ml313.git
    cd ml313
    pip install .
